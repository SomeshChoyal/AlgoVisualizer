# AlgoVisualizer

AlgoVisualizer is a project focused on visualizing algorithms through interactive applications. Currently, it includes a visualizer for the N-Queens problem and an Android application for enhanced usability.

## N-Queens Visualizer

The N-Queens Visualizer is designed to help users understand the N-Queens problem by providing a visual representation of the algorithm's execution. The visualizer displays the placement of queens on a chessboard and demonstrates the backtracking algorithm used to solve the problem.

### Features

- Interactive N-Queens visualization
- Step-by-step algorithm execution
- Dynamic board size adjustment
- Real-time UI updates

## Installation

1. **Download the APK**:
   - Visit the [Releases](https://github.com/SomeshChoyal/AlgoVisualizer/releases) page of this repository.
   - Download the APK file for the latest release.

2. **Install the APK**:
   - On your Android device, go to `Settings` > `Security` (or `Apps`).
   - Enable installation from unknown sources if it's not already enabled.
   - Open the downloaded APK file and follow the on-screen instructions to install the app.

3. **Run the App**:
   - Open the app from your app drawer and start using it.
